# 🌐 Guía de Interacción con Servicios: FTP (File Transfer Protocol)

Servicio estándar para la transferencia, mediación y monitoreo de archivos.
## Clientes y Conexión

* **Conexión mediante el cliente FTP estándar:** `ftp 192.168.2.142`.
* **Conexión directa al puerto mediante Netcat:** `nc -v 192.168.2.142 21`.
